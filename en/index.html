What is Hazelcast IMDG?

An open-source distributed in-memory data storage used as a cache, distributed synchronization tool and/or message broker. It provides out of the box support for various data structures such as maps, lists, sets or synchrozination primitives and offers computing capabilities over them.

Swiss-Army knife out of the box

Without any further configuration, you get distribution storage and computing cluster having all the data structures ready. Data distribution across the cluster, automatic migration of data in case of node failure, redistribution of workload in case of scaling up - all out of the box right from the beginning.

Use from any language

Client server architecture with clients implemented in various popular languages makes accessing the cluster possible from any application you need. We also provide integrations with countless third party tools to take another piece of work from you.
